# Support

## Project Status

The Day After Toolkit is in active public build-out.

This is personal research and development work by Enrico Piovesan, done on personal time.

Questions, bug reports, and contribution discussions should generally go through GitHub issues and pull requests unless the topic is security-sensitive.

## Where to Ask

- Use GitHub issues for bugs, feature requests, and implementation discussion
- Use pull requests for code review and change discussion
- Use the project board for delivery tracking: [GitHub Project 2](https://github.com/users/enricopiovesan/projects/2)

## Security Issues

Do not report security issues publicly in GitHub issues.

Follow [SECURITY.md](SECURITY.md) for private reporting expectations.
