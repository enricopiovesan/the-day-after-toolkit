# Code of Conduct

This repository is meant to be useful to readers, contributors, and maintainers.

## Expected Behavior

- be respectful and direct
- assume good intent, but stay technically rigorous
- focus criticism on code, docs, and decisions, not on people
- keep discussion useful for future readers of the repo

## Unacceptable Behavior

- harassment or personal attacks
- discriminatory language or behavior
- deliberate disruption of discussions, issues, or pull requests
- bad-faith technical argument meant to score points instead of improve the work

## Scope

This applies to:

- issues
- pull requests
- discussions around the repository
- other public spaces used to coordinate work on this project

## Reporting

If something needs to be reported, open a private channel with the maintainer rather than escalating publicly in a thread that is already going off track.

## Enforcement

Maintainers may remove comments, close threads, or block participation when behavior is clearly harmful to the project or its contributors.
